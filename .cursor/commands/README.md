# Cursor Commands — Memory Bank

Команды интегрированы из [cursor-memory-bank](https://github.com/vanzan01/cursor-memory-bank) (v0.8) и работают с папкой `memory-bank/`.

## Команды

| Команда | Назначение |
|--------|------------|
| `/van` | Инициализация задачи, определение сложности, маршрутизация |
| `/plan` | План реализации (уровни 2–4) |
| `/creative` | Варианты дизайна (архитектура, UI/UX) |
| `/build` | Реализация по плану |
| `/reflect` | Рефлексия по выполненной работе |
| `/archive` | Архив задачи, сброс контекста |

## Быстрый старт

1. Открой **Chat** (Ctrl+L или Cmd+L) — команды по `/` работают только в Chat, не в Composer.
2. В поле ввода чата введи `/van` и описание задачи, например:  
   `/van Исправить баг с публикацией в Дзен`
3. Дальше следуй подсказкам: для уровня 2–4 будет предложен `/plan`, затем при необходимости `/creative`, затем `/build` → `/reflect` → `/archive`.
