# Тренировка разметки Дзена (Playwright)

Временный файл: все типы разметки редактора Дзен на отдельных блоках. Использовать для отладки вставки через Playwright.

---

## Список типов разметки (по порядку)

| № | Тип | Тег / блок | Описание |
|---|-----|------------|----------|
| 1 | Заголовок 1 | h1 | Обычно только в поле «Заголовок» статьи; в теле может не отображаться |
| 2 | Заголовок 2 | h2 | Подзаголовок секции |
| 3 | Заголовок 3 | h3 | Подпункт |
| 4 | Заголовок 4 | h4 | Мелкий заголовок |
| 5 | Абзац | p | Обычный текст |
| 6 | Маркированный список | ul + li | Буллиты |
| 7 | Нумерованный список | ol + li | Нумерация |
| 8 | Цитата | blockquote | Выделение цитаты |
| 9 | Ссылка | a внутри p | Текст со ссылкой |
| 10 | Картинка | блок image | Отдельный блок (path + caption) |

---

## HTML для вставки (копировать в буфер и вставлять в тело статьи по одному)

Каждый блок ниже — один блок контента. Вставка: фокус в редактор → Ctrl+V.

### 1. Заголовок 1 (h1)
```html
<h1>Заголовок 1 (h1)</h1>
```

### 2. Заголовок 2 (h2)
```html
<h2>Заголовок 2 (h2)</h2>
```

### 3. Заголовок 3 (h3)
```html
<h3>Заголовок 3 (h3)</h3>
```

### 4. Заголовок 4 (h4)
```html
<h4>Заголовок 4 (h4)</h4>
```

### 5. Абзац (p)
```html
<p>Абзац (p) — обычный текст параграфа.</p>
```

### 6. Маркированный список (ul)
```html
<ul>
<li>Пункт маркированного списка 1</li>
<li>Пункт маркированного списка 2</li>
<li>Пункт маркированного списка 3</li>
</ul>
```

### 7. Нумерованный список (ol)
```html
<ol>
<li>Пункт нумерованного списка 1</li>
<li>Пункт нумерованного списка 2</li>
<li>Пункт нумерованного списка 3</li>
</ol>
```

### 8. Цитата (blockquote)
```html
<blockquote>Цитата (blockquote) — выделенный блок цитаты.</blockquote>
```

### 9. Ссылка (a в p)
```html
<p>Текст со <a href="https://example.com">ссылкой (a)</a> внутри абзаца.</p>
```

### 10. Картинка
В нашем формате — отдельный блок `type: "image"` с `path` и `caption`. В редакторе вставляется через кнопку «Загрузите файл» / «Фото», не через HTML.
