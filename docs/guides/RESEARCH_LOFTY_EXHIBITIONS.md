## Итог

## Рекомендации для дальнейшего поиска

1. **RUN IT** — уточнить у организаторов (AGIMA) или в RUN IT Connect/партнёрских материалах, входила ли Lofty в список партнёров 2024/2025 и есть ли контакт/упоминание компании.
2. **SWITCH Connect** — при наличии доступа поиск по «Lofty», «Kupikod», «MyBid» и по российским/СНГ компаниям.
3. **Affiliate World (Dubai/Europe)** — на странице экспонентов с фильтрами проверить вручную по названиям и по странам.
4. **LinkedIn** — события (Events), в которых указана компания Lofty или сотрудники с должностями Founder/CEO/Co-Founder; посты с хештегами мероприятий (например, #SWITCH, #DMEXCO, #SigmaEurope).
5. **Фото/отчёты с выставок** — поиск по «Lofty», «Kupikod», «MyBid» в Google Images и на сайтах мероприятий (галлереи, отчёты партнёров).
6. **Подкасты/интервью** — запросы вида «Lofty AdTech интервью», «Kupikod founder», «MyBid CEO» на русском и английском.
