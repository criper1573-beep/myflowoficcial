# Parsers — парсеры и мониторы

Скрипты деплоя и конфигурации для парсеров и мониторов (Telegram и др.).

Мониторинг Telegram-канала: ловит новые сообщения с заданным заголовком и шлёт уведомления ботом.

**Деплой:**
```bash
# Из корня проекта (или PROJECT_DIR=/path/to/project)
bash parsers/telegram_monitor/setup.sh
```
