# Статья: Тренды офисных пространств 2026

**Файл статьи (JSON):** `article_trends_office_2026.json`  
**Не публиковать:** в файле выставлено `"publish": false`.

## Параметры генерации

- **Тема:** Тренды офисных пространств 2026
- **Проект:** flowcabinet
- **Ключевые запросы:** из `wordstat_top_queries.csv` (ремонт офисов, ремонт офиса под ключ, дизайн офиса, стоимость ремонта офисов и др.)
- **Фактура:** собрана по открытым источникам (тренды дизайна офисов 2025–2026, гибридные пространства, wellness, экологичность)

## Как опубликовать позже

1. Подготовить или сгенерировать изображения (cover + block1–4).
2. Скопировать содержимое `article_trends_office_2026.json` в `article_to_publish.json` либо запустить публикацию с флагом/путём к этому файлу, если скрипт поддерживает.
3. Выставить `"publish": true` при готовности.
4. Запустить: `npm run zen:publish` (если публикуется из `article_to_publish.json`) или соответствующую команду для этого файла.
